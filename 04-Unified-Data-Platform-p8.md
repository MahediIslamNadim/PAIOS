# Learning Activity Model (LAM)

## Purpose

The Learning Activity Model records learning behavior, study sessions, practice, revision, assessments, and progress.

It transforms raw study activity into structured evidence that supports analytics and personalized recommendations.

Learning activities become inputs for the Learning Intelligence Engine.

# Summary

The Learning Activity Model provides a structured representation of study behavior.

By connecting sessions, activities, revisions, quizzes, and progress, it creates an evidence-rich foundation for adaptive learning and long-term analytics.
