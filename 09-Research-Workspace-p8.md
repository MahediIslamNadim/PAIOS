# Research Pipeline & Automation

## Purpose

Research Pipelines automate common research workflows connecting paper discovery, reading, experimentation, and publication.

---

# Predefined Research Pipelines

New Paper Pipeline

Discover -> Import -> Read -> Annotate -> Summarize -> Link to knowledge -> Add to literature review

Experiment Pipeline

Design -> Setup -> Execute -> Collect data -> Analyze -> Document -> Publish results

Publication Pipeline

Draft -> Review -> Submit -> Track -> Revise -> Publish -> Track impact

---

# Pipeline: Paper Reading Workflow

Paper Discovered (search, recommendation, import)

↓

Import Metadata (DOI, arXiv, BibTeX)

↓

Download PDF

↓

AI Generates Summary

↓

User Reads Paper

↓

Annotate (highlights, notes)

↓

AI Extracts Key Information

↓

Link to Knowledge Graph

↓

Add to Literature Review

↓

Update Reading Progress
