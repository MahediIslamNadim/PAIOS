# Productivity Pipelines & Automation

## Purpose

Productivity Pipelines automate daily workflows connecting planning, execution, review, and adjustment.

Automation ensures consistent productivity habits without manual overhead.

---

# Predefined Productivity Pipelines

Daily Planning Pipeline

AI generates plan -> Review calendar -> Check goals -> Schedule time blocks -> Set priorities -> Start day

Evening Review Pipeline

Log completed tasks -> Review goals -> Journal entry -> Plan tomorrow -> Check habits

Weekly Review Pipeline

Analyze analytics -> Review goals -> Adjust plans -> Set priorities for next week -> Schedule deep work
