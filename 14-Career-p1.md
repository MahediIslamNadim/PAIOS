# Career Overview

## Purpose

The Career module helps users manage their professional development, job applications, and career progression.

---

# Scope

Portfolio Builder

Certificates & Credentials

CV/Resume Builder

Interview Tracker

Internship Tracker

Career Roadmap

---

# Entity: career_entries

Purpose

Stores career-related items.

Fields

entry_id

entry_type (portfolio, certificate, resume, interview, internship, roadmap)

title

description

date

url (portfolio link, certificate link)

status

notes

created_at

---

# Portfolio Builder

Showcase projects and achievements.

Supported Items

Projects (linked from Engineering/Programming)

Certificates

Publications (linked from Research)

Open Source Contributions

Work Experience

Skills (linked from Learning)

---

# CV/Resume Builder

Generate CV/resume from profile data.

Features

Template Selection

Section Editor

Auto-fill from modules

Export PDF

Multiple versions

---

# Interview Tracker

Track job interview pipeline.

Pipeline Stages

Applied -> Phone Screen -> Technical -> On-site -> Offer -> Accepted/Declined

Each stage has date, notes, follow-up.

---

# Career Roadmap

Long-term career planning.

Features

Goal Setting (next role, skills to acquire)

Timeline

Skill Gap Analysis

Course Recommendations (linked to Learning)

Progress Tracking
